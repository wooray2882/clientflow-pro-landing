# Examples Folder

This folder contains visual references and example files for Ray's workflow preferences.

---

## How to Add Examples

1. **Screenshots/Images**: Drop `.png`, `.jpg`, or `.webp` files here
2. **Workflow exports**: Add `.json` exports from n8n
3. **Diagrams**: Add system diagrams or flowcharts
4. **Templates**: Add reusable templates

---

## Naming Convention

Use descriptive names:
- `dashboard-layout-preferred.png` - Your preferred dashboard style
- `sticky-note-organization.png` - How you arrange sticky notes
- `workflow-webhook-standard.json` - Standard webhook workflow template
- `system-architecture-example.png` - System design you like

## Describing Your Examples

After adding an image, update the relevant `.md` file with a description:

```markdown
## Example: dashboard-layout-preferred.png

**What this shows:**
- Dark theme with sidebar navigation
- Cards arranged in 3-column grid
- Gold accent for primary actions
- Minimal, spacious layout

**Key elements to replicate:**
- Card border radius: 12px
- Spacing between cards: 24px
- Background color: #0a0a0b
```

This helps the AI understand your preferences from the visual.
